1. 陣列array

裡面可以儲存很多的值 ,用來方面管理資料, 例如同型態的多筆資料

(1)定義陣列:var 陣列名稱=[值1, 值2, 值3, ...];
舉例:存放學生成績
var scores = [80, 30, 89, 70];
document.write(scores);
:80,30,89,70

(2)取得陣列內容:陣列名稱[要取得的值的順序位子]
舉例:取得第2位學生資料
var scores = [80, 30, 89, 70];
document.write(scores[1]);
:30

(3)修改陣列內容:陣列名稱[要改值的順序位子] = 要改成的值
舉例:修改第二位學生的成績
var scores = [80, 30, 89, 70];
scores[1] = 56;
document.write(scores);
:80,56,89,70

(4)觀念
# 順序是從0開始
# 存放的型態可以是字串, 陣列, 布林值
var YuSay = ["girl", 77, true]; document.write(YuSay);
:girl,77,true

(5)取得陣列長度:陣列名稱.length
var scores = [80, 30, 89, 70];
document.write(scores.length);
:4

1. 函式function

預先寫好的程式碼, 當我們今天進行呼叫, 函式內的程式碼才會被執行,
因此, 函式可以分成兩個部分, 第一個部分是函式的"定義", 再來是函式的"呼叫"

(1)函式定義:
function 函式名稱(){
要運行的程式內容;
}
舉例:定義一個名叫hello的函式
function hello(){
document.write("你好");
}
//如果只有定義, 則函式的內容不會被執行

(2)呼叫函式:函式名稱();
舉例:呼叫hello函式
function hello(){
document.write("你好");
}
hello();
你好
//此時內容才會被執行

(3)函式可以有傳入值
function 函式名稱(傳入值){
要運行的程式內容;
}
舉例:設定ok函式的傳入值為ans, 並且進行呼叫, 輸入傳入值給函式
function ok(ans){
document.write("Ans=" + ans);
}
ok("rue");
Ans=rue

(5)傳入值也可以設為多個
小測驗:add函式, 可輸入兩個數並且顯示相加結果
function add(num1, num2){
ans = num1 + num2;
document.write("ans="+ans);
}
add(5, 7);
ans=12

(6)函式的回傳:return
可以將回傳的值, 覆蓋原先呼叫的位子
舉例:add用回傳的方式印出
function add2(num1, num2){
ans = num1 + num2;
return ans;
}
document.write("ans2="+ add2(4, 5));
ans2=9

3. if判斷

(1)if判斷:
if(條件成立){
要執行的程式碼;
}
舉例:如果肚子餓, 印出我就去吃飯
var hungry = true;
if(hungry){
document.write("我就去吃飯");
}
:我就去吃飯

(2)if..else
if(條件成立){
要執行的程式碼;
}
else{
條件不成立時, 要執行的程式碼;
}
舉例:如果下雨, 我就開車, 否則我就走路
var rainy = true;
if(rainy){
document.write("我會開車");
}
else{
document.write("我會走路");
}
:我會開車

(3)if..else if..else
if(條件成立){
要執行的程式碼;
}
else if(另一條件成立){
要執行的程式碼;
}
else{
條件都不成立時, 要執行的程式碼;
}
舉例:如果考100分得100塊, 考60分得60塊, 不然就是10塊
var score = 100;
if(score == 100){
document.write("考100分得100塊");
}
else if(score >= 60){
document.write("考60分得60塊");
}
else{
document.write("得10塊");
}
:考100分得100塊
如果前面的條件已符合, 就不會再去執行後面的程式

(4)/&&(且:有否為否)、||(或:有真為真)、!(反面)
var score = 100;
var rainy = true;
if(score==100 && rainy){
document.write("考100分得100塊且我會開車");
}
else if(score==100 || rainy){
document.write("考100分得100塊或是我會開車");
}
else if(score==100 || !rainy){
document.write("考100分得100塊或是我會走路");
}
:考100分得100塊且我會開車

(5)小測驗:max_num函式可以傳入三個數, 並且回傳最大值
function max_num(num1, num2, num3){
if(num1 >num2 && num1 > num3){
return ("最大值為="+ num1);
}
else if(num2 >num1 && num2 > num3){
return("最大值為="+ num2);
}
else{
return("最大值為="+ num3);
}
}
document.write(max_num(3,6,4));
:最大值為=6
