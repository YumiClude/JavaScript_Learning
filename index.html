4. 如何使用字串[字串用法]

(1)如果我們想要印出雙引號",只要在雙引號前面加上\即可
var phrase = "hello\"lala";
document.write(phrase);
:hello"lala

(2)印出字串,變數相加(連接), 利用"+"
var phrase = "hello\"lala";
var my_name = "YuSay";
document.write(phrase + my_name);
:hello"lalaYuSay

(3)算字串長度:變數名稱.lenght
var phrase = "hello\"lala";
document.write(phrase.length);
:10

(4)字串全部變成大寫:變數名稱.toUpperCase()
var phrase = "hello\"lala";
document.write(phrase.toUpperCase());
:HELLO"LALA

(5)字串全部變成小寫:變數名稱.toLowerCase()
var phrase = "hello\"lala";
document.write(phrase.toLowerCase());
:hello"lala

(6)回傳字串第x+1位的文字:變數名稱.charAt(x)
var phrase = "hello\"lala";
document.write(phrase.charAt(1));
:e

觀念:JavaScript的世界中, 順序是從0開始算的->0123456789

(7)回傳某個內容是字串中第幾位:變數名稱.indexOf()
var phrase = "hello\"lala";
document.write(phrase.indexOf("l"));
:2
注意:他所回傳的數值, 是從0開始算的
如果是裡面沒有的字元, 則會回傳"-1"
var phrase = "hello\"lala";
document.write(phrase.indexOf("r"));
:-1
提醒:如果是重複的內容, 則會回傳最前面出現的那一個的位子

(8)回傳字串第a位到第b位的內容:變數名稱.substring(a, b)
var phrase = "hello\"lala";
document.write(phrase.substring(2, 6));
:llo"

5. 如何使用數字[數字用法]

(1)印出數字
document.write(34.5);
:34.5

(2)觀念:
[1]運算符號:+(加),-(減),*(乘),/(除),%(取餘數)
[2]規則:規則同一般數學概念:順序為先乘除後加減,括號內會優先運算
[3]另外, 如果變數的型態是數字, 也可以直接拿來運算
var num = 7;
document.write("變數加7:");
document.write(num + 7);
變數加7:14
[4]提醒:提醒:如果跟字串類型的變數一起印出, 則結果會變成字串
document.write("變數加7,但是跟字串寫在一起:"+ num + 7);
變數加7,但是跟字串寫在一起:77

(3)絕對值:Math.abs(變數名稱)
num = -6;
document.write("變數為-6的絕對值:");
document.write(Math.abs(num));
變數為-6的絕對值:6

(4)找x,y,x..多個數字的最大的那個數:Math.max(x,y,x)
num = -6;
document.write("找3,7,-1,4,7,5以及變數num之中最大值:");
document.write(Math.max(3,7,-1,4,7,5,num));
找3,7,-1,4,7,5以及變數num之中最大值:7

(5)找x,y,x..多個數字的最小的那個數:Math.min(x,y,x)
num = -6;
document.write("找3,7,-1,4,7,5以及變數num之中最小值:");
document.write(Math.min(3,7,-1,4,7,5,num));
找3,7,-1,4,7,5以及變數num之中最小值:-6

(6)四捨五入:Math.round(數字or變數名稱)
num = 3.6;
document.write("變數num為3.6做四捨五入:");
document.write(Math.round(num));
變數num為3.6做四捨五入:4

(7)//x乘以y次方:Math.pow(x,y)
document.write("3的5次方:");
document.write(Math.pow(3,5));
3的5次方:243

(8)開根號:Math.sqrt(數字or變數名稱)
document.write("36開根號:");
document.write(Math.sqrt(36));
36開根號:6

(9)回傳隨機值:Math.random()
document.write("產生一個隨機值:");
document.write(Math.random());
產生一個隨機值:0.6840878024342225(每次都會不同)
如果我們希望這個隨機值是0~10之間, 則可以把Math.random()*10,
接著在進行四捨五入Math.round(Math.random()*10)
document.write("產生一個0~10的隨機值:");
document.write(Math.round(Math.random()*10));
產生一個隨機值:7(每次都會不同)
